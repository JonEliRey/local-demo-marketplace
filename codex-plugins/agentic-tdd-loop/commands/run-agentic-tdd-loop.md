description: Run the adopting organization agentic TDD loop with implementation, review, and simplification gates.
---

Use the `agentic-tdd-loop` skill to turn the current request into a RED -> GREEN -> REVIEW -> FIX -> SIMPLIFY execution loop.

Required behavior:

1. Identify the smallest testable behavior.
2. Write or update the failing test first.
3. Implement the smallest passing change.
4. Run focused and full verification gates.
5. Request adversarial review for meaningful changes.
6. Apply only material fixes.
7. Summarize changed files, verification, and remaining approval gates.
